Find the largest number among three numbers entered by user

Step 1: Start
Step 2: Declare variables a, b, c
Step 3: Read variables a,b and c
Step 4: If a > b and c then display a 
        If b > a and c then display b
        If c > a and b then display c
Step 5: Stop


pseudo code:
    a, b, c: int32
    read(a, b, c)
    if a > b and c:
        print(a)
    if b > a and c:
        print(b)
    if c > a and b:
        print(c)
